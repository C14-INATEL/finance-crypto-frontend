# finance-crypto-frontend

Plataforma de gestão financeira que centraliza o controle de investimentos em criptoativos, permitindo que o usuário realize o cadastro seguro e gerencie sua carteira de forma inteligente. Através da integração com apis de mercado em tempo real, o sistema consolida o histórico de transações (compras e vendas) e calcula automaticamente o patrimônio atualizado e a rentabilidade de cada ativo, exibindo tudo em um dashboard.


Iniciar projeto
```
ng serve -o
```
Rodar teste
```
ng test
```
